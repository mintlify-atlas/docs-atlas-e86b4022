### GitHub Contributions Chart API

Contribution data is provided by the [sallar/github-contributions-chart](https://github.com/sallar/github-contributions-chart) project, which offers a public API for retrieving GitHub user contributions.

**API Endpoint:**
```
https://github-contributions.vercel.app/api/v1/{username}
```

This API returns contribution data including:
- Available years of contribution history
- Daily contribution records with dates and intensity levels
- Contribution counts and statistics
