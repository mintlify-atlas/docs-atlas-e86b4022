      Quick start
    </h2>
    <p className="text-base text-gray-600 dark:text-gray-400">
      Get up and running with GitHub Contributions Chart in minutes
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Download the project from GitHub to get started.

      ```bash
      git clone https://github.com/Steema/TeeChart-VCL-GitHub-Contributions.git
      cd TeeChart-VCL-GitHub-Contributions
      ```
    </Step>

    <Step title="Open in your IDE">
      Open `GitHubContributions.dproj` in Delphi or C++Builder. Make sure you have TeeChart VCL installed.

      <Note>TeeChart VCL is required to compile and run this application. Visit [Steema TeeChart](https://www.steema.com/product/vcl) to learn more.</Note>
    </Step>

    <Step title="Build and run">
      Compile the project and launch the application. Enter a GitHub username (e.g., "sallar") and click **Get Contributions** to see the visualization.

      The application will fetch data from the public GitHub Contributions API and render an interactive chart showing contribution patterns over time.
    </Step>
  </Steps>
</div>
